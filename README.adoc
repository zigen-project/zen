= Zen

The reference compositor implementation of Zigen protocol

== Build & Install

[source, shell]
----
$ git clone https://github.com/zigen-project/zen.git
$ cd zen
$ meson build
$ ninja -C build install
----

== Run

----
$ mkdir /tmp/.xdg
$ XDG_RUNTIME_DIR=/tmp/.xdg zen
----
